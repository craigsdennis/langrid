Language Grid Service Manager

The Service Manager is a web-based tool to manage the Language Grid for the Language Grid Users and the Operator.
This tool allows easy management of user information, user access, language/computer resources and language services. Each role in the Language Grid has different range of control.

This site is compatible with Internet Explorer 7.0 and Firefox.

All Language Grid Users

  • All Language Users access the information
    • - News 
      • You can access the operation history of the Language Grid, such as registration/suspending/resuming/deletion of language resources and computation resources.
    • - Language Grid Users 
      • You can access the Language Grid User List to check the registered user groups.
    • - Language Resources 
      • You can access a list of all the language resources on the Language Grid.
    • - Language Services 
      • You can access a list of all the language services on the Language Grid.
    • - Computation Resources 
      • You can access a list of all the computation resources on the Language Grid.
    • - Service Types 
      • You can access a list of all the service types on the Language Grid.

  • Password Change

    To ensure the security and fair use of the Language Grid, Language Grid Users can change the password for their ID.


  • User Profile Setting

    Language Grid Users can edit setting of their profile information.

  • Registration and Deletion of Language Resources
    You can register your language resources and add the resources' profile information (such as a name, type, target languages of the resources) and release to other Users. Also you can delete the resources and profile information from the Language Grid.

  • Setting of Copyright and License Information
    You can put the URL describing the copyright or license information of your language resources in the Language Grid to notify the Users. The information can be displayed with each response message to the Language Grid Users.

  • Registration and Deletion of Language Services
    You can register your language services and add the services' profile information (such as a name, type, target languages of the services) and release to other Users. Also you can delete the services and profile information from the Language Grid.

  • Suspending and Restart of Language Services
    You can suspend or resume an access to your language services from the Users. Also you can send notification of the suspension or resuming to the Users.

  • Access Control of Language Services
    You can set access control on your language services. By each user, you can control access to your service by limiting number of access per year/month/day. Likewise, you can limit maximum data amount per access per year/month/day by each user. Also you can set access right to your services by user, and that allows you to give access permission only to particular users or block access from other particular users.

  • Monitoring of Language Services
    You can check the statistical information on the use of your language services, which includes total number of access to your service per year/month/day, or the information about who accessed when to your services. Also, you can check the registration information of the users of your language services. Please note that the statistical information mentioned above does not include each message sent by the users or the personal information about the users.

  • Retrieval of Language Services
    You can retrieve available language services.

Language Grid Operator

  • Account Registration and Deletion
    The Operator can create or delete an account for Language Grid User.

  • Monitoring of Language Resource Misuse
    Setting conditions of use, the Operator monitors usage of all language resources on the Language Grid to detect misuse of language resources.

  • Monitoring of Language Resource Usage
    The Operator checks the statistical information including total number of access to all language resources on the Language Grid per year/month/day or full access log about who accessed when. Also, the Operator has access to the registration information of the users to know who exactly used the resources.

  • Access Control of Unauthorized Users
    The Operator can remove access right of a user who may be misusing language resources.
